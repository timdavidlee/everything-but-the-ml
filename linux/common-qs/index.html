    How to answer common Linux system Questions

    What's running on this machine?

    A great tool to use

    # MacOSx
brew install htop

# RHEL | Centos | red-hat | amazon
sudo yum install -y htop

# ubuntu | debian | mint
sudo yum apt-get install -y htop

    how much space is left on the drive?
